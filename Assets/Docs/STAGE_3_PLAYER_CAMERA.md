# Stage 3: Player Movement And Third-Person Camera

This stage adds only placeholder player traversal in `VillageScene`.

## Controls
- `WASD` / arrow keys: move.
- `Left Shift`: run.
- Hold right mouse button and move mouse: orbit camera.

## Scene Objects
- `Reynard_Player`: capsule placeholder with `CharacterController` and `PlayerController`.
- `ThirdPersonCamera`: camera with `ThirdPersonCamera` follow/orbit script.
- `VillageBlockoutGround`: large test ground plane.
- `VillageMovementTestArea`: colored markers for movement/camera orientation testing.

## Not Included Yet
- Combat.
- Quests.
- Inventory.
- Interactions.
- Save/load.
