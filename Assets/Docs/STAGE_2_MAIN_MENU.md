# Stage 2: Main Menu And Settings Stub

This stage builds only the main menu surface. It does not implement player movement, combat, quests, inventory, or saves.

## Menu Buttons
- New Game: loads `VillageScene`.
- Continue: shows a "No save file yet" status message.
- Settings: opens the settings panel.
- Exit: quits in build or logs in editor.

## Settings Stub
- Volume slider.
- Music toggle.
- Resolution dropdown placeholder.
- Graphics dropdown placeholder.
- Apply button stores values in `PlayerPrefs`.
- Back button returns to the main panel.
