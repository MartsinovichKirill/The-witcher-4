## Итог

Таблицы 3.7-3.22 в пояснительной записке присутствуют. По текущему состоянию проекта они покрывают основные функции игры: меню, настройки, управление, NPC, квесты, бой, инвентарь, крафт, сохранение, загрузку и концовку.
